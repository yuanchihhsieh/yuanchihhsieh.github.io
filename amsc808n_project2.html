AMSC808N Project 2

This page collects three related numerical experiments from AMSC808N: a variational neural network solver for the committor problem, a PINNs solver for the Allen-Cahn equation, and a spectral baseline used for reference and error analysis.

Focus: operator learning and scientific machine learning for transition-path analysis and PDE solving, with side-by-side comparisons between neural and classical numerical methods.

Committor PINNs Spectral Baseline

Variational Neural Network for the Committor

This project solves the committor equation on the rugged Muller potential with high-dimensional noise using a neural network derived from the variational formulation. The committor \(q(x)\) measures the probability that a stochastic trajectory reaches set \(B\) before set \(A\), making it a natural object for identifying transition regions.

\[ q(x)=\mathbb{P}_x \left\{ \tau_B < \tau_A \right\} \]

The learned solution is validated through committor analysis near the level set \(q=0.5\). Points close to the estimated transition surface are sampled, short stochastic trajectories are launched from each point, and the empirical hitting probability is compared against the ideal value of \(0.5\).

  • The \(q=0.5\) isocommittor is extracted in the \((x_1, x_2)\)-plane to identify the transition region.
  • Monte Carlo trajectory tests check whether the learned surface is balanced between the two metastable sets.
  • The empirical histogram peaks near \(0.477\), which is consistent with a well-placed transition surface under finite-sample noise.

PINNs for the Allen-Cahn Equation

This part of the project uses physics-informed neural networks to solve the 1D Allen-Cahn initial-boundary value problem. The training objective combines PDE residual loss with initial and boundary condition losses so the neural network approximates the full space-time solution directly.

\[ u_t - 0.0001u_{xx} + 5u^3 - 5u = 0 \]

The network is trained for 2000 epochs and then evaluated against a spectral reference solution. The comparison shows that the PINNs model captures the phase-separation structure and interface evolution well, while also offering fast evaluation once training is complete.

  • The final comparison reports MAE \(\approx 1.04 \times 10^{-1}\) and RMSE \(\approx 2.70 \times 10^{-1}\).
  • The predicted solution range remains close to the reference range, indicating stable learning of the phase structure.
  • Time-slice comparisons show slightly smoother transitions than the reference solver while preserving the correct overall dynamics.

Spectral Baseline for Allen-Cahn

The spectral baseline provides the reference solution used to evaluate the PINNs model. A Fourier spectral discretization on a uniform grid gives a fast and accurate numerical benchmark, making it useful both for validation and for understanding where the neural approximation smooths or deviates from the baseline.

The saved spectral dataset contains the full space-time solution on a dense grid with 256 spatial points and 1001 time levels. This reference solver is substantially faster to generate than the PINNs training loop and serves as the main accuracy baseline for error visualization.

  • The spectral solver supplies the benchmark arrays used for pointwise comparison across the whole space-time grid.
  • Heatmap comparisons make the agreement and residual structure immediately visible.
  • The workflow highlights a practical tradeoff: classical solvers are efficient to generate for fixed settings, while trained neural surrogates are attractive for rapid post-training inference.
